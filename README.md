## Konovo backend task

> Features:

- OpenAPI support
- Bearer token auth
- List products
- Get single product
- Filter products by name, category, brand, price range
- Pagination and sorting
- Basic product processing

> Run app:

`uv run fastapi dev`

> Run typechecking:

`uv run ty check`

> Run linter and formatter:

`uv run ruff check`
